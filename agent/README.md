# Agent Instructions

This directory contains instructions and standards that the agent (Antigravity) should follow when working on this project.

### Organization

- `instructions/`: Contains specific rule documents.
- `workflows/`: Contains automated or semi-automated task procedures.

### Current Standards

1. [Commenting Standard](instructions/comments.md): Rules for function-level documentation.
2. [Responsive Design](instructions/responsive-design.md): Ensuring 100% responsiveness across layouts.
3. [API Design Principles](instructions/api-design.md): Ensuring a clean and maintainable codebase.
